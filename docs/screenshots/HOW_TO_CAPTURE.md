# How to Capture App Screenshots for README

The README references 8 screenshot files in this folder.  
Take them from the interactive prototype in one step.

## Steps

1. Open `docs/ui_prototype.html` in **Chrome** or **Edge**
2. Open DevTools → press `F12`
3. Click **Toggle device toolbar** (Ctrl+Shift+M) → select **iPhone 14 Pro** (390×844)
4. Use the **screen buttons** at the bottom of the prototype to navigate to each screen
5. Press `Ctrl+Shift+P` → type **Capture screenshot** → save as the filename below

## Required Files

| Filename | Screen to Capture | Button to Click |
|----------|-------------------|-----------------|
| `screen-login.png` | Login screen | 🔐 Login |
| `screen-register.png` | Register screen | 📝 Register |
| `screen-forgot.png` | Forgot Password screen | 🔑 Forgot Pwd |
| `screen-home.png` | Home dashboard | 🏠 Home |
| `screen-chat.png` | Chat screen (HR example) | 💬 Chat · HR |
| `screen-clarify.png` | Low-confidence clarify screen | ❓ Clarify |
| `screen-admin.png` | Admin document panel | 🛡️ Admin |
| `screen-upload.png` | Upload document modal | ⬆ Upload |

## Tips

- Capture **only the phone frame** — crop out the purple background and nav buttons
- Target size: **390 × 844 px** (or scale to ~200px wide for README display)
- Save as PNG for crisp text
- All 8 files go in this folder: `docs/screenshots/`
